    Oi Controle

    Web / Mobile

    Melhorias na página do plano de celular Oi Controle

    Cliente: oi.com.br Empresa: vml.com 
    Sobre

    A Oi é uma das maiores empresas de telefonia móvel do Brasil, com mais de 30 milhões de clientes cadastrados no segmento de mobilidade pessoal.

    Fui contratado pela vml.com para fazer parte do time de e-commerce responsável pelo plano Oi Controle, trabalhando em conjunto com o time de negócios e desenvolvimento.

    Alinhamento

    Objetivos e Planejamento

    ❮ ❯

    Para iniciar o projeto foi feito uma reunião com a equipe de UX, Negócios e Desenvolvimento para analisar e definir as principais necessidades do projeto.

    Objetivos

    - Melhorar conversão para o funil de vendas
    - Otimizar a usabilidade da página mobile
    - Reaproveitar componentes existentes

    O objetivo do projeto era identificar oportunidades que pudessem melhorar a conversão de usuários para o processo de assinatura 
    Descoberta

    Problemas e Oportunidades

    ❮ ❯
    processo

    Analisamos dados de fluxo de conversão, mapa de calor, testes de usabilidade e vídeos de navegação para mapear os principais pontos de atenção na jornada do usuário, desde a página página inicial até a conclusão do processo de assinatura.

    Descoberta

    - Saída de usuários do fluxo de conversão
    - Padrões de cliques nos cards de planos
    - Diferenciais do produto sem destaque
    - Serviços adicionais sem explicação
    - Página muito extensa

    Identificamos um padrão de comportamento onde o usuário abandonava a página de assinatura e retornava para a página de produto em busca de mais informações 
    Exploração

    Hipóteses e Rascunhos

    ❮ ❯
    Idéias

    - Destaque para os diferenciais do plano
    - Aproveitar a interação com os cards
    - Ordenar planos pelo menor valor
    - Otimizar o conteúdo da página
