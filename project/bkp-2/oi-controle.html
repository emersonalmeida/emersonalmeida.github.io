Oi Controle

Web / Mobile

Melhoria na usabilidade da página do plano de celular Oi Controle

Cliente: oi.com.br
Consult: vml.com 
Sobre

A Oi é uma das maiores empresas de telefonia móvel do Brasil, com mais de 30 milhões de clientes cadastrados nos planos de mobilidade pessoal.

Fui contratado pela vml.com para fazer parte do time de e-commerce responsável por atualizar as campanhas da marca e propor melhorias para o e-commerce.

Alinhamento

Objetivos e Planejamento

Para iniciar o projeto foi feito uma reunião com a equipe de UX, Negócios e Desenvolvimento para analisar e definir as principais necessidades do projeto.

Objetivos

- Melhorar conversão para o funil de vendas
- Otimizar a usabilidade da página mobile
- Reaproveitar componentes existentes 

Função

- Identificar pontos de melhoria
- Analise de mapa de calor
- Teste de usabilidade
- Analise competitiva
- Prototipagem
- Validação

O objetivo do time era propor melhorias focando na conversão de usuários para o processo de assinatura do plano 
Pesquisa

Analise Inicial

Foram analisados dados de mapa de calor, gravações de tela, fluxo de conversão e testes de usabilidade para mapear os principais pontos de atenção na jornada do usuário

O plano Oi Controle é indicado para clientes que buscam os melhores benefícios da assinatura mas com um valor acessível no final do mês.

❮ ❯

O plano Oi Controle é indicado para clientes que buscam os melhores benefícios do produto com um bom custo-benefício.

Descoberta

- Abandono do fluxo de assinatura
- Padrões de cliques nos cards de planos
- Diferenciais do produto sem destaque
- Serviços adicionais sem explicação
- Conteúdo secundário ocupando espaço de tela
- Página muito extensa em relação aos concorrentes

Oportunidade
