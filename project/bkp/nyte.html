    Nyte App

    Web / Mobile

    Aplicativo de festas e eventos locais

    Cliente: Hospital do Amor Empresa: Ydreams Global 
    Sobre

    A Oi é uma das maiores empresas de telefonia móvel do Brasil, com mais de 30 milhões de clientes cadastrados nos planos pré-pago, controle e pós-pago.

    Um dos projetos foi a melhoria da página mobile do plano Oi Controle, voltado para clientes que buscam benefícios intermediários com um valor fixo mensal.

    Fui contratado pela vml.com para fazer parte do time de UX do e-commerce, dando suporte ao time de negócios e de desenvolvimento.

    Objetivo, Expectativas e Processos

    Alinhamento

    Para iniciar o projeto foi feito uma reunião com a equipe de UX, Negócios e Desenvolvimento para analisar e priorizar as principais necessidades de cada área.

    Objetivos

    - Melhorar conversão para o funil de vendas
    - Identificar oportunidade e validar hipóteses
    - Reaproveitar componentes existentes

    Processo

    - Criação - UX
    - Aprovação - Negócios
    - Desenvolvimento - Tecnologia
    - Homologação - QA
    - Deploy - Teste A/B
